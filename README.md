This is a simple frontend portion of project Blog created using Nextjs. Add .env file referencing .env.example file.

## Getting Started

First, Install all the dependency: 
```bash
npm install 
# or
yarn install
```

And then run the development server:
```bash
npm run dev
# or
yarn dev
```
